Getting started with Mixture

Posted: Mon Nov 26 2012

I’m currently working on a series of posts and screencasts for Mixture which will focus on individual features and better describe how you can use them.

Here’s my current release plan…

  1. Overview, briefly introducing all that Mixture has to offer
  2. Project structure & templating
  3. Preprocessors (including debugging with the chrome extension)
  4. Models (how you can use real or test data with Mixture)
  5. Optimisation & debugging
  6. Local development (hostnames & multi-device testing, inc. live reload)
  7. Pushing a project online
    Each screencast will be added to the documentation site http://docs.mixture.io/tutorials as they become available.

If you are interested in this series then please follow @TeamMixture for updates.

If there is a particular feature you would like us to cover, please let us know.
